Les revenus en France

Description du projet

Etude sur les revenus en France à partir des données impôt sur le revenu par collectivité territoriale

Comment ça marche ?

Pour le moment, seul la barre de recherche est fonctionnelle, mais elle permet d'accéder à pas mal d'informations pour chaque département à présent. Mais de futures fonctionnalités viendront s'ajouter pour compléter cette application

Lesquelles ?

Le projet n'est encore qu'à ses débuts, mais les idées fuses dans tous les sens...

  • De nombreuses cartes de France
  • Des cartes par département (cartographie des villes)
  • Comparaison entre départements
  • De nouvelles "Data Viz"
Mais étant seul sur ce projet pour le moment, cela prend du temps.

A propos

Ce projet est en fait un ensemble de sous projets qui ont été réalisés avec l'aide de différentes technologies (JavaScript, C#..). Ces projets sont accessibles sur la plateforme github

Les résultats obtenus (visibles à travers cette application web), ne sont pas garantis "sans bugs ou erreurs" de ma part. Mais si vous repérez un problème, n'hésitez pas à me contacter.

Technologies utilisées

  • JSON.NET : création d'objets JSON
  • jQuery
  • jQuery Typeahead
  • Bootstrap (Twitter) et glyphicons
  • Chartist-JS
  • Numeral.JS
  • Et sûrement d'autres à venir...
